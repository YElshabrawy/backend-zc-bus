## Project Description

-   The project is a mobile application that provides a bus service for Zewail City students.
-   The application will provide the following features:
    -   User authentication.
    -   Bus routes and schedules.
    -   Bus tracking.
    -   Notifications.
    -   User feedback.
    -   User profile.
